# Privacy Policy for AuthFlow App

**Last Updated:** January 3, 2026

## Introduction

AuthFlow ("we," "our," or "the app") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your personal information when you use our Android application.

## 1. Information We Collect

### 1.1 Personal Information You Provide
When you create an account, we collect:
- **Full Name**: Used for personalization and display
- **Email Address**: Used for authentication and communication
- **Password**: Encrypted and securely stored via Firebase Authentication

### 1.2 Automatically Collected Information
- **Device Information**: Device type, operating system version, unique device identifiers
- **Usage Data**: App features accessed, session duration, crash reports
- **Advertising Data**: Ad impressions, clicks, and interactions (via Google AdMob)

### 1.3 Firebase Services
We use **Google Firebase** services, which may collect:
- Authentication data (email, UID)
- Database records (user profile information)
- Analytics data (app usage patterns)

## 2. How We Use Your Information

### 2.1 Primary Purposes
- **Authentication**: To create and manage your account
- **User Experience**: To personalize your app experience
- **Security**: To protect against unauthorized access
- **App Improvement**: To analyze usage and fix bugs

### 2.2 Advertising
We use **Google AdMob** to display advertisements. AdMob may collect:
- Device advertising ID (AAID/IDFA)
- IP address
- Device information
- Ad interaction data

For more details, see Google's Privacy Policy: https://policies.google.com/privacy

---

## 3. Data Storage and Security

### 3.1 Firebase Realtime Database
- Your profile data (name, email) is stored in **Google Firebase Realtime Database**
- Data is encrypted in transit (HTTPS/TLS)
- Access is restricted to authenticated users only

### 3.2 Firebase Authentication
- Passwords are hashed and never stored in plain text
- Authentication tokens are securely managed by Firebase

### 3.3 Security Measures
- Industry-standard encryption (TLS 1.2+)
- Secure Firebase security rules (users can only access their own data)
- Regular security audits and updates
